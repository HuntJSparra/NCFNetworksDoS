Ping Flood

By sending as many pings as possible, an attacker with greater bandwidth use all of the victim's resources, preventing a legitimate user from connecting.

Code

Unix Terminal

sudo ping -f -s 51064 {VICTIM IP}

...that's it


Explaination

On Unix devices, the ping command already comes with a flood option: "-f". Using this command request admin priviledges, thus sudo is also required unless the user is already the root. "-s" simply specifies the size of the packet. The larger the packet, the fewer packets are needed to consume all of the victim's bandwidth.
